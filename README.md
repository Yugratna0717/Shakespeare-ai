# 🎭 Shakespeare AI

An AI-powered Shakespeare tutor built with Streamlit and Gemini.

## Features

- Explain Shakespeare scenes line by line
- Convert Elizabethan English into modern English
- Simple web interface
- Powered by Gemini API

## Installation

```bash
pip install -r requirements.txt
streamlit run app.py
```

## Environment Variables

Create a `.env` file:

```env
GEMINI_API_KEY=your_api_key_here
```

## Tech Stack

- Python
- Streamlit
- Gemini API
- python-dotenv
